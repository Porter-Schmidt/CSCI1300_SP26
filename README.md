# CSCI1300_SP26

## Week 1
* Introduced to the first program in C++ called HelloWorld.
* Friday activity includes HelloBlanks, where students are asked to use words from a word bank to complete the Hello World script.

## Week 2
* Friday activity focused on problem solving, encouraging students to write out pseudocode in the comments to guide their implementation of solving how to find the area of a trapezoid. 
* At the end of class, students were asked to modify their original trapezoid script to ask for user input rather than hard coding values.
